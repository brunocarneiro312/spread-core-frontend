Como Utilizar

Instalação

Os componentes podem ser baixados através do github .

Para clonar o repositório, siga os seguintes passos:

  1. Entre no prompt de comandos do Windows e navegue até o diretório onde o projeto será baixado 
    ( + r) cmd.exe
  2. Digite o comando: 
    git clone https://github.com/brunocarneiro312/spread-core-frontend.git
  3. Após clonar o repositório, é necessário instalar as dependências: 
    npm install
  4. Execute o comando 
    npm run dev

Essa mesma interface será aberta localmente em sua máquina.
