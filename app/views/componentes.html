Componentes

Frontend

Os componentes de tela desenvolvidos utilizam as seguintes tecnologias:

  • EcmaScript6 (Javascript)
  • AngularJS
  • HTML5
  • CSS3

Os componentes desenvolvidos podem utilizar as tecnologias acima mencionadas em conjunto ou de forma isolada.

Accordion

O componente Accordion é um componente estilo "porta sanfonada". Quando um item é selecionado, os demais são minimizados.

Overlay

O componente Overlay é um componente estilo "cortina". Quando utilizado como um elemento HTML, é adicionado ao body, preenchendo toda a tela. Quando utilizado com um atributo, o elemento o qual recebeu o atribulo será preenchido.

  • Accordion
  • Carousel
  • Modal
  • Overlay
  • Table
